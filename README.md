# Aphorox

Aphorox es una aplicación de escritura tipo diario enfocada en la simplicidad, fluidez y registro natural del pensamiento.

## ✍️ Características

- Escritura libre sin interrupciones
- Registro automático por día
- Timestamps con doble Enter
- Guardado automático
- Estilos de texto (italica, tipografía)
- Zoom dinámico

## 🧠 Filosofía

Aphorox no busca ser un editor complejo, sino un espacio donde escribir se sienta natural.

## ⚙️ Tecnologías

- Python
- PyQt5

## 🚀 Cómo ejecutar

python3 main.py
